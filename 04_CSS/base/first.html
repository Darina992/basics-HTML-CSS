Форматы файлов.

.doc или .DOC(от англ. document) — расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение «.doc» часто использовалось для обозначения текстовых файлов, содержащих текст без форматирования, позже стало использоваться для двоичных форматов с разметкой.
Документ Microsoft Word
Portable Document Format (PDF) — межплатформенный открытый формат электронных документов, изначально разработанный фирмой Adobe Systems с использованием ряда возможностей языка PostScript.
Документ PDF
ZIP — популярный формат архивации файлов и сжатия данных без потерь.Файл в этом формате имеет расширение .zip Архив ZIP может содержать один или несколько файлов и каталогов, которые могут быть сжаты разными алгоритмами.
